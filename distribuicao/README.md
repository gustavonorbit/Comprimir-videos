# Project Codename (Beta)

Um programa simples para deixar seus vídeos menores e mais fáceis de compartilhar — reduz o tamanho do arquivo sem complicar.

> O nome final do projeto ainda não foi decidido. "Project Codename" é só um nome provisório.

## Como instalar (Mac)

1. Baixe o arquivo **`Project Codename.dmg`** desta pasta.
2. Dê dois cliques nele para abrir.
3. Vai abrir uma janela com o ícone do app e um atalho da pasta **Aplicativos** lado a lado — arraste o ícone do app pra cima do atalho.
4. Pronto, o app já está instalado. Pode fechar essa janela e abrir o app normalmente pela pasta Aplicativos (ou pelo Launchpad/Spotlight).

> Prefere baixar um `.zip` em vez do `.dmg`? Também tem o arquivo `Project Codename.zip` nesta pasta — é mais leve, bom pra conexões mais lentas. Basta descompactar e arrastar o `.app` pra pasta Aplicativos.

### "App de desenvolvedor não identificado"

Na primeira vez que abrir o app, o Mac provavelmente vai avisar que ele é de um "desenvolvedor não identificado" e não vai deixar abrir com um clique normal. Isso é esperado — o app ainda não tem certificado da Apple. Pra abrir mesmo assim:

- Clique com o **botão direito** (ou control+clique) no ícone do app → escolha **Abrir** → confirme **Abrir** de novo na janela que aparecer.
- Isso só é necessário na primeira vez.

## Como usar

1. Abra o app.
2. Clique em **Selecionar Vídeo** e escolha o arquivo.
3. Deixe as opções no padrão (**Balanceado**) ou ajuste qualidade/resolução se quiser.
4. Clique em **Comprimir Vídeo** e aguarde.
5. O vídeo comprimido aparece na mesma pasta do vídeo original.

## Atenção — versão Beta

Este é um build de teste, feito para ser compartilhado rapidamente com poucas pessoas. Ainda **não é a versão final**:

- não precisa instalar mais nada além do próprio app;
- pode ter bugs;
- a interface e os recursos ainda vão mudar.

Encontrou algum problema ou tem sugestão? Me avise!
